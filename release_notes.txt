# Smart Window Tool v1.3.0

本版本重新设计了主窗口和悬浮菜单，并集中修复透明度、菜单定位、窗口焦点和自启动流程中的问题。

## 重点更新

- 主窗口与悬浮菜单采用统一的现代化视觉样式。
- 自动跟随 Windows 深色与浅色主题。
- 优化整窗透明度处理，减少背景发白和文字颜色异常。
- 菜单首次显示即保持在当前屏幕工作区内，避免越界和位置跳动。
- 窗口信息界面支持按 `Esc` 关闭，并改善键盘焦点处理。
- 完善任务栏边界、多显示器和窗口定位逻辑。
- 开机自启动改用计划任务，并改进管理员权限切换与单实例同步。
- 增加自启动、操作版本和菜单边界定位测试，共 20 项测试。
- README 预览图已更新为新版界面。

## 下载与运行

- 下载 `SmartWindowTool-v1.3.0.exe` 后直接运行。
- 需要 Windows 10 或 Windows 11。
- 需要安装 .NET 8 Windows Desktop Runtime。
