# Smart Window Tool v1.0.0 正式版发布

🚀 这是 Smart Window Tool 的首个正式稳定版本，基于 WPF 和 .NET 8 重构，为你带来前所未有的极客级窗口管理体验。

## 🌟 核心亮点特性

- **极致流畅的现代 UI**：全局采用 Windows 11 Fluent Design（圆角、Mica 毛玻璃、阴影、深浅色自适应），彻底告别上古 Win32 弹窗。
- **独创的滚轮控制**：
  - `Ctrl + Shift + 滚轮`：无极调节窗口透明度
  - `Ctrl + Alt + 滚轮`：平滑调节窗口高度
  - `Shift + Alt + 滚轮`：平滑调节窗口宽度
- **老板键与隐藏**：强制将任意不支持隐藏的程序最小化到系统托盘，并为其自动生成专属托盘图标。
- **多屏防打扰**：支持一键将窗口迁移至下一显示器；支持进程黑名单过滤，防止游戏中误触快捷键。
- **智能边缘贴靠**：强迫症福音，一键自动吸附到最近的屏幕边缘。
- **纯净单文件**：整个程序被打包为单一的 `.exe` 可执行文件，无需安装繁琐的运行库，开箱即用。
