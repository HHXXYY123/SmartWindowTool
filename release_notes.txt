# Smart Window Tool v1.3.1

本版本修复关闭“开机自启动”时可能错误提示“计划任务删除失败”的问题。

## 修复内容

- 修复任务不存在时异常类型识别错误，避免把已删除的任务误判为删除失败。
- 删除计划任务改为幂等操作，任务已经不存在时直接视为成功。
- 优先使用 Windows 任务计划程序接口删除，失败时再回退到 `schtasks`。
- 回退删除后会复查任务是否确实不存在，避免误报成功。
- 区分当前用户任务与旧版任务的删除错误，提示信息更加明确。
- 增加真实任务计划程序集成测试，全部 23 项测试通过。

## 下载与运行

- 下载 `SmartWindowTool-v1.3.1.exe` 后直接运行。
- 需要 Windows 10 或 Windows 11。
- 需要安装 .NET 8 Windows Desktop Runtime。
