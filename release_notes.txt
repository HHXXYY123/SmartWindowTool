# Smart Window Tool v1.3.2

本版本调整开机自启动路径行为：从哪个 EXE 开启自启动，下次登录就直接运行哪个 EXE。

## 修复内容

- 计划任务直接记录当前正在运行的 `SmartWindowTool.exe` 完整路径。
- 不再把程序复制到固定的 `%ProgramFiles%\SmartWindowTool` 目录。
- 不再把自启动路径重定向到托管副本。
- 更新 README，明确移动或删除已绑定的 EXE 会导致自启动失效。
- 增加任务创建参数回归测试，确保始终保留当前 EXE 路径。
- 全部 24 项测试通过。

## 使用说明

- 切换自启动程序位置前，请先完全退出正在运行的旧实例。
- 从目标 EXE 启动程序，然后关闭并重新开启“开机自启动”。
- 单实例机制生效时，再次双击其他位置的 EXE 只会激活已有实例。

## 下载与运行

- 下载 `SmartWindowTool-v1.3.2.exe` 后直接运行。
- 需要 Windows 10 或 Windows 11。
- 需要安装 .NET 8 Windows Desktop Runtime。
